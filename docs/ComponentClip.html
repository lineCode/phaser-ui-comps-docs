Class: ComponentClip

ComponentClip

Component view group builds itself with given jfsl-generated config object; Supports state switching. Best if controlled by ComponentPrototype instance.

new ComponentClip(scene, config, textures, isChildComponent)

Parameters:
Name Type Argument Description
scene Phaser.Scene

Phaser scene to create component at
config ComponentConfig
textures Array.<String>

Array of texture names, where component should find its texture frames
isChildComponent Boolean <nullable>
